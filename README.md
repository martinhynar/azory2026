# Doporučené tempo

Na Azorech se nevyplatí spěchat.

Počítejte s častými zastávkami:

- fotografie,
- květiny,
- výhledy,
- oceán,
- odpočinek.

Průměrná rychlost na stezkách bude kolem **2,5 km/h**.

---

# Bezpečnost

- zůstaňte na značených trasách,
- nevstupujte za zábrany u útesů,
- po dešti počítejte s blátem,
- sledujte počasí,
- mějte dostatek vody.

# Termální koupání

## Terra Nostra Park

⭐⭐⭐⭐⭐

Nejznámější termální bazén.

### Teplota

cca 38 °C

### Voda

Železitá.

Nezapomeňte:

- tmavé plavky,
- starší ručník.

Doporučená doba koupání:

20–30 minut.

⭐⭐⭐⭐☆

- menší bazény,
- večerní atmosféra.

Teplota:

37–39 °C
